\# 2026-04-23 TIL



깃 명령어 익히기.

* cd: 폴더이동
* git branch: 현재 어떤 브랜치인지 확인.
* git checkout -b test-branch: test-branch라는 브랜치를 만들면서 그쪽으로 이동. -b는 "새로 만들어라" 는 옵션.
* touch branch-test.md: 빈 파일 생성.
* notepad branch-test.md: 메모장으로 파일 열기.
* git add .: 방금 만든 파일 커밋 준비 상태로 올림.   . 은 "변경된것 전부"라는 뜻.
* git commit -m "test-branch에서 파일 추가": 준비된 파일을 저장하면서 무슨작업했는지 메모 남김. 이 커밋은 test-branch에만 존재함.
* git checkout main: main 브랜치로 이동.
* ls: 파일 목록 확인.
* merge: 두 브랜치 합치기(내가있는 브랜치로).
