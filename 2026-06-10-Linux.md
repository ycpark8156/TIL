# 2026-06-10 Linux 서버 실습

## 오늘 배운 것

### 파일 공유 (NFS, Samba)

- NFS 서버 설치 및 공유 폴더 설정
- exports 파일로 공유 대상 네트워크 지정
- NFS 클라이언트에서 mount로 연결
- Samba로 윈도우에서 네트워크 드라이브 연결

### 메일 서버 (Postfix, Dovecot)

- Postfix : 메일 보내는 서버 (SMTP)
- Dovecot : 메일 받아서 보관 (IMAP/POP3)

### DNS 서버 (BIND9)

- 정방향 존 : 도메인 → IP
- 역방향 존 : IP → 도메인
- named-checkconf, named-checkzone으로 설정 검사

## 핵심 개념

- NFS : 리눅스 서버끼리 폴더 공유 (구글 드라이브랑 비슷한 개념)
- Samba : 윈도우 ↔ 리눅스 폴더 공유
- SMTP : 메일 보내는 프로토콜
- IMAP/POP3 : 메일 받는 프로토콜
- DNS : 도메인을 IP로 변환 (전화번호부 같은 개념)

## 헷갈렸던 것

- tee vs echo : tee는 파일 저장 + 화면 출력, echo는 화면 출력만
- 절대경로 vs 상대경로
- systemctl restart vs enable
