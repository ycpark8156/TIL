# 2026-06-09 Linux 기본 명령어

## 오늘 배운 것

### 파일/디렉토리 탐색
```bash
pwd	#현재 위치 출력
ls	#파일 목록 보기
ls -al 	#숨김파일 포함 상세 목록
cd ~	#홈 디렉토리로 이동
cd ..	#상위 디렉토리로 이동
```

### 파일/디렉토리 생성
```bash
mkdir 폴더명	#폴더 생성
touch 파일명	#파일 생성
```

## 웹서버 실습 (Apache, Nginx)

### 오늘 한 것
- Apache 설치, 가상호스트 설정, 보안 헤더 적용
- Nginx 설치, 서버블록 설정, 리버스 프록시 실습

### 핵심 개념
- 웹서버 : 브라우저 요청 받아서 파일 돌려주는 것
- 가상호스트 : 서버 하나에서 여러 사이트 운영
- 리버스 프록시 : Nginx가 앞에서 받아서 뒤 서버로 전달
- Apache랑 Nginx 둘 다 80번 포트라 동시에 못 켬

### 헷갈렸던 것
- 설정 파일 만들어도 활성화 따로 해야 함
- reload vs restart 차이
